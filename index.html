NodeMCU with Built-in OLED

This build uses a NodeMCU ESP8266 board with an integrated OLED — no wiring needed! Super neat for quick debugging, sensor readouts, or just seeing if your code is alive.

Here’s how I brought it to life:

  1. Install the U8g2 library in Arduino IDE.
  2. Upload the code below — it includes an I2C scanner and a "Hello" message.
  3. Check your Serial Monitor for I2C feedback, then enjoy the display!

// OLED Hello World + I2C Scanner for NodeMCU

#include <Wire.h>
#include <U8g2lib.h>

// NodeMCU I2C Pins
#define SDA_PIN D2
#define SCL_PIN D1

// OLED setup (SSD1306, 128x64)
U8G2_SSD1306_128X64_NONAME_F_HW_I2C u8g2(U8G2_R0, U8X8_PIN_NONE, SCL_PIN, SDA_PIN);

void setup() {
  Serial.begin(115200);
  Wire.begin(SDA_PIN, SCL_PIN);

  // I2C scan (just in case)
  Serial.println("Scanning I2C bus...");
  byte error, address;
  for (address = 1; address < 127; address++) {
    Wire.beginTransmission(address);
    error = Wire.endTransmission();
    if (error == 0) {
      Serial.print("Device found at 0x");
      if (address < 16) Serial.print("0");
      Serial.println(address, HEX);
    }
  }

  u8g2.begin();
}

void loop() {
  u8g2.clearBuffer();
  u8g2.setFont(u8g2_font_ncenB08_tr);
  u8g2.drawStr(0, 24, "Hello from HamHacks!");
  u8g2.sendBuffer();
  delay(1000);
}

⚠️ If you get a blank screen, check your I2C address in the Serial Monitor. Some displays default to 0x3C, others to 0x3D. This code helps figure that out before you tear your hair out.

Once it's talking, it's rock solid. Try changing the message or font for fun!
