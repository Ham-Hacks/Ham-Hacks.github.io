Blinking an LED using GPIO

This is the “Hello World” of hardware — and it never gets old.

import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BCM)
GPIO.setup(17, GPIO.OUT)

try:
    while True:
        GPIO.output(17, GPIO.HIGH)  # LED on
        time.sleep(1)
        GPIO.output(17, GPIO.LOW)   # LED off
        time.sleep(1)
except KeyboardInterrupt:
    GPIO.cleanup()

Boom — your LED blinks. Double-check your wiring if it doesn’t.
