Blinking an LED using GPIO

Alright, let's kick things off with the classic “Hello, World!” of hardware: blinking an LED. It's a rite of passage. You get your Pi wired up, fire off a little Python, and boom — that tiny light winks at you like, “Yep, we’re alive.”

You’ll need just a few basics to get going: a Raspberry Pi, a breadboard, an LED, a 330Ω resistor (or close to it), and some jumper wires. We’ll use GPIO pin 17 for this one — easy to remember, easy to find. Just refer to the pinout image above if you're unsure.

Now here's the code. It’s pretty simple — we just set up the pin as an output and toggle it on and off in a loop. 

import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BCM)
GPIO.setup(17, GPIO.OUT)

try:
    while True:
        GPIO.output(17, GPIO.HIGH)  # LED on
        time.sleep(1)
        GPIO.output(17, GPIO.LOW)   # LED off
        time.sleep(1)
except KeyboardInterrupt:
    GPIO.cleanup()

When you run this, your LED should start blinking on and off once per second. If not, double-check your wiring — especially the resistor placement and whether the LED is facing the right way (long leg goes to GPIO!).

That’s it. Super simple, but super satisfying. From here, you can play around — speed up the blink rate, add more LEDs, or maybe try using a button to control it. This is where the fun really starts 🔧😄
